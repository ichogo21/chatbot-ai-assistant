# 🤖 Claude AI - Content Brainstorming Chatbot

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/)
[![Anthropic API](https://img.shields.io/badge/API-Anthropic%20Claude-purple.svg)](https://www.anthropic.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Asisten chatbot interaktif berbasis CLI (Command Line Interface) yang dirancang khusus untuk membantu konten kreator melakukan *brainstorming* ide konten secara cepat dan terarah menggunakan kecerdasan buatan dari Claude API (Anthropic).

---

## ✨ Fitur Utama
* **Interactive CLI:** Antarmuka terminal yang ramah pengguna dan interaktif untuk berkirim pesan langsung dengan Claude.
* **Smart Content Brainstorming:** Prompt dasar yang dioptimalkan untuk memancing ide kreatif, sudut pandang konten unik, dan struktur skrip.
* **Secure Environment:** Menggunakan integrasi `.env` untuk memastikan keamanan API Key pengguna selama proses pengembangan.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **AI Model:** Anthropic Claude API (`anthropic` python SDK)
* **Configuration:** `python-dotenv` (manajemen variabel lingkungan)
