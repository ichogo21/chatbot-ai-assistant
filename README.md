# 🤖 Groq AI - Content Brainstorming Chatbot (Llama 3)

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue.svg)](https://www.python.org/)
[![Groq API](https://img.shields.io/badge/API-Groq%20Llama%203-orange.svg)](https://groq.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Asisten chatbot interaktif berbasis CLI (Command Line Interface) yang dirancang khusus untuk membantu konten kreator melakukan *brainstorming* ide konten secara kilat menggunakan teknologi akselerasi inferensi dari Groq API dan model Llama 3.1.

---

## ✨ Fitur Utama
* **Super Fast Inference:** Didukung oleh teknologi Groq untuk hasil respons instan dalam hitungan milidetik.
* **Interactive CLI with Memory:** Antarmuka terminal dengan kemampuan mengingat konteks obrolan sebelumnya (*chat history*).
* **Secure Config:** Menggunakan integrasi `.env` untuk mengamankan API Key dari publikasi tidak sengaja.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **AI Model:** Meta Llama 3.1 (via Groq API SDK)
* **Configuration:** `python-dotenv`
